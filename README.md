# 🚀 Circle

A platform to find team members and mentors for your projects.

`Frontend Stack`

- Typescript
- Next.js
- TailwindCSS

## Installation 🔧

Install dependencies

```
$ yarn

```

Start the Development Server

```
$ yarn dev

```
